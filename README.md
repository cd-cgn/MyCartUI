# ECommApp
Enterprise Scale ECommerce Application

# User Interface
- React UI
[UserExperience]
- Admin UX : 
Login -> Add/Remove Sellers -> Logout

- Seller UX : 
Login / SignUP -> Add/Remove Products -> Logout

- Customer UX : 
Login / SignUP -> 
Products Page Add -> Cart Page Add/Remove/Order -> Order Page Cancel/Delivered 
-> Logout
